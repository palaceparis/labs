3.3 Power, effect size and sample size

These questions are adapted from section 7.2 of Nordmann, Cleland-Woods, and McAleer (2020)

3.3.1 Calculating effect size given a test statistic

The effect size (Cohen’s d) is |μ1−μ0|/σ|μ1−μ0|/σ. According to Cohen (1988), the effect size is small (.2 to .5), medium (.5 to .8) or large (> .8). In the context of a one sample tt-test we can estimate Cohen’s d from t0=x¯−μ0s/n√t0=x¯−μ0s/n, the observed test statistic, as d=t0/n−−√d=t0/n.

You run a one-sample tt-test and discover a significant effect, R reports the test statistic to be 3.24 with 25 degrees of freedom and p<.05p<.05. Using the above formulas, calculate dd and determine whether the effect size is small, medium or large.

n = 25 + 1
d = 3.24/sqrt(n)
d
## [1] 0.6354163

The effect size, 0.64, is categorised as a medium effect size.

Using pwr.t.test(), find how many people per group ( nn ) you would need to detect an effect size of d=.4d=.4 with 80% power at the 5% significance level in a two sample tt-test with a two sided alternative hypothesis. How does this change as the effect size changes (e.g. from 0.1 to 0.8)? Plot the sample size required to achieve a power of 80% against effect size.

library(pwr)
pwr.t.test(d = 0.4, power = 0.8, sig.level = 0.05, alternative = "two.sided", 
    type = "two.sample")
## 
##      Two-sample t test power calculation

##     alternative = two.sided

## NOTE: n is number in *each* group

We would need 100 people in each group to achieve a power of at least 80%.

d_vals = seq(0.1, 0.8, 0.01)
req_n = NULL
for (i in seq_along(d_vals)) {
    req_n[i] = pwr.t.test(d = d_vals[i], power = 0.8, sig.level = 0.05, 
        alternative = "two.sided", type = "two.sample")$n
}
plot(d_vals, req_n, type = "l")
abline(v = 0.4, lty = 2)
abline(h = 100, lty = 2)

As the effect size decreases, the sample size required to achieve 80% power increases dramatically.
